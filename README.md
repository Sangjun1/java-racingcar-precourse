# 자동차 경주 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능 목록
- 0-9 사이의 랜덤 기능 구현
- 각 자동차에 이름을 부여하도록 기능 구현
- 사용자에게 몇번을 이동할지 입력 기능 구현
- 4이상일 경우 전진하는 기능 구현
- 자동차 경주게임 승리 메소드 구현
- 입력값의 체크로직 구현
